# TBCV Reports Directory

This directory contains various reports generated during TBCV development and operation.

## Directory Structure

### `/sessions/`
Session progress reports from development phases:
- `P1-P8*.md` - Phase 1-8 implementation session reports
- `autonomous_session*.md` - Autonomous development session reports
- `session_summary*.md` - General session summaries
- `final_session*.md` - Final session reports
- `bugfix_session*.md` - Bug fix session reports

### `/coverage/`
Test coverage JSON files:
- `coverage_*.json` - Coverage reports from different phases
- Generated by pytest with `--cov` flag

### `/archive/`
Historical reports and artifacts:
- Old JSON reports and analysis files
- Deprecated report formats
- Historical analysis documents

## Generating New Reports

Reports are typically generated by:
- Development sessions (manual markdown)
- Test suite with coverage: `pytest --cov=. --cov-report=json`
- System analysis scripts in `scripts/maintenance/`

## Cleanup Policy

- **Keep**: Current phase reports, latest completion summaries
- **Archive**: Older session reports (after major milestones)
- **Remove**: Duplicate or superseded reports
- **Never commit**: Generated coverage HTML reports (use JSON only)
